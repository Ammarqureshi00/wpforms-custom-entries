== Description ==
WPForms Custom Entries is a plugin that allows you to customize the entries submitted through WPForms. This plugin provides additional functionality to manage and display form entries in a more flexible way.

== Installation ==
1. Upload the `wpforms-custom-entries` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to WPForms > Custom Entries to configure the plugin settings.

== Frequently Asked Questions ==
= Does this plugin require WPForms? =
Yes, WPForms Custom Entries requires the WPForms plugin to be installed and activated.

= Can I customize the display of form entries? =
Yes, this plugin allows you to customize the display of form entries through various settings and options.
